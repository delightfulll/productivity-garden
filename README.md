# Productivity Garden

A beautiful productivity app that helps you organize your tasks into three categories:

- Watering Tasks: Tasks that move the needle
- Sunlight Tasks: Tasks that keep your goals alive
- Composting Tasks: Tasks that are extraneous but necessary

## Features

- Beautiful UI with smooth animations
- Task categorization
- Drag and drop task reordering
- Task completion with confetti animation
- Responsive design

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Framer Motion for animations
- React Router for navigation

## Getting Started

1. Clone the repository

```bash
git clone [your-repo-url]
```

2. Install node dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```

## Project Structure

- `/src/components` - Reusable components
- `/src/screens` - Main page components
- `/src/App.css` - Global styles
- `/src/index.css` - Base styles
